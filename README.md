## 📖 About The Project

Tickitz is a modern, user-friendly movie ticket booking platform that provides a seamless experience for moviegoers. Built with pure HTML, CSS, and JavaScript, this project showcases a responsive design that works flawlessly on both desktop and mobile devices.

The application fetches real movie data from The Movie Database (TMDB) API, providing users with up-to-date information about current and upcoming films. With an intuitive interface and smooth navigation, Tickitz aims to revolutionize how people book movie tickets online.

## 🚀 Features

- **🎭 Current Movie Listings** - Browse through popular movies currently showing
- **🔮 Upcoming Movies** - Explore movies coming soon to theaters
- **🏷️ Genre Tagging** - Movies categorized by genres for easy browsing
- **👑 Recommendations** - Highlighted movies with high ratings
- **📱 Responsive Design** - Seamless experience across all devices
- **🔔 Newsletter Subscription** - Stay updated with the latest releases
- **🔐 Authentication** - User registration and login functionality
- **🎨 Modern UI/UX** - Clean, intuitive interface with smooth animations

## 🛠️ Built With

## 💻 Getting Started

### Prerequisites

- A modern web browser
- A code editor (VS Code, Sublime Text, etc.)
- Basic knowledge of HTML, CSS, and JavaScript

### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/mhakimsaputra17/weekly-task3.git
   ```

2. Navigate to the project folder
   ```sh
   cd weekly-task3
   ```

3. Open `index.html` in your browser or use a live server extension

### API Configuration

This project uses the TMDB API. To use it with your own API key:

1. Sign up for an API key at [TMDB](https://www.themoviedb.org/documentation/api)
2. Replace the API key in `index.js`
   ```js
   Authorization: "Bearer YOUR_API_KEY"
   ```
